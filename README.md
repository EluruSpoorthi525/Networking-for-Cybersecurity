# Networking-for-Cybersecurity

This repository contains beginner-friendly notes on networking concepts that every cybersecurity and GRC professional should understand.

## Topics Covered

- OSI Model
- TCP/IP Model
- DNS
- HTTP
- HTTPS
- VPN
- Firewalls

## Why This Repository?

Networking is the foundation of cybersecurity. Understanding how data travels across networks helps security professionals identify risks, implement controls, and improve security posture.

## Skills Learned

- Networking Fundamentals
- Security Concepts
- Internet Protocols
- Traffic Flow
- Secure Communication
