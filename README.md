# Networking Fundamentals for Cybersecurity

## Overview

Networking is one of the most important foundations of cybersecurity. Before learning penetration testing, security operations, cloud security, governance, risk, and compliance (GRC), it is essential to understand how devices communicate across networks and how attackers exploit network vulnerabilities.

This repository provides detailed notes on fundamental networking concepts with cybersecurity-focused explanations. Every topic is written in beginner-friendly language and includes practical examples, security relevance, interview questions, and learning resources.

---

## Objectives

- Learn networking from the ground up.
- Understand how communication occurs over the Internet.
- Build a strong cybersecurity foundation.
- Prepare for cybersecurity certifications.
- Strengthen knowledge required for GRC, SOC, and Network Security roles.
- Create a professional GitHub portfolio.

## What You Will Learn

After completing this repository, you will understand:

- How data travels across a network
- The purpose of each OSI layer
- How TCP and UDP differ
- How DNS resolves domain names
- How HTTP and HTTPS work
- The SSL/TLS encryption process
- How VPNs create secure communication tunnels
- Different types of firewalls
- Common networking ports and protocols
- Basic network security principles

## Why Networking Matters in Cybersecurity

Networking knowledge is essential for:

- Governance, Risk, and Compliance (GRC)
- Security Operations Center (SOC)
- Penetration Testing
- Incident Response
- Digital Forensics
- Cloud Security
- Network Security Engineering

Without networking fundamentals, understanding cyber attacks, vulnerabilities, and security controls becomes difficult.
